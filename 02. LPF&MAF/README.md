Filter : 특정 주파수를 차단시키거나 통과시킴으로서 원하는 주파수를 볼 수 있도록 하는 것으로 Inductor(L)와 Capacitor(C)를 조합하여 구현이 가능하다.
- Inductor  : 주파수 값이 올라갈수록 얼마나 고주파의 흐름을 방해하는가 (주파수가 올라갈수록 통과가 잘 됨)
- Capacitor : 주파수 값이 올라갈수록 얼마나 고주파의 흐름을 원할 하게 하는가
